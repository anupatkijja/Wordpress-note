# Note

## Plugin
```
Cache - WP Super Cache -  แคชหน้าต่าง ๆ ของ WordPress ไว้ ทำให้โหลดเว็บไซต์ได้เร็วขึ้นหลายเท่า
Minify & Lazyload - Autoptimize - ย่อขนาดไฟล์ CSS / Javascript แบบอัตโนมัติ ทำให้โหลดหน้าเว็บได้เร็วขึ้น
Secuirty - AIO Security
Image Optimize - ShortPixel Image Optimizer
Backup - Updraft
WP Rocket

Contact Form 7 - เอาไว้สร้างแบบฟอร์มสำหรับติดต่อเรา ปรับแต่งได้เยอะ ใช้งานง่าย
Types - (แนะนำ) เพิ่ม Custom Post Type, Custom Taxonomy, Custom Field เพิ่มความสามารถให้ WordPress กลายเป็น CMS ในพริบตา
MobileMonkey’s WP-Chatbot - ปลั๊กอินตัวนี้เข้ามาช่วยในส่วนของการแชทผ่านเว็บไซต์ โดยใช้ตัว Facebook Messenger และยังสามารถเพิ่มข้อมูลของเขาโดยอัตโนมัติ สามารถทำให้เก็บข้อมูลลูกค้าได้ด้วย
MonsterInsights - ปลั๊กอินตัวนี้ช่วยให้ Google Analytics สามารถมองเห็นข้อมูลเวิร์ดเพรสของคุณได้
Yoast SEO - นี่คือปลั๊กอินที่ดีที่สุดใน WordPress สำหรับการทำ SEO โดยตัวนี้จะช่วยตรวจเช็คประสิทธิภาพของบทความของคุณ โดยจะตรวจเช็คและแนะนำให้ปรับปรุงบทความเพื่อให้ง่ายต่อการติดอันดับ SEO
```
